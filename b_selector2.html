		ex) input[type=text]{}

자식 선택자와 후손 선택자

		자식 선택자: 자식 요소를 선택하는 선택자.
		ex)선택자>선택자
		
		후손 선택자: 후손요소를 선택하는 선택자.
		ex)선택자 선택자

반응선택자

		사용자의 움직임에 따라 HTMl Element를 선택하는 선택자
		1. 사용자가 클릭한 요소 선택 : 선택자:active{}
		2. 사용지기 마우스를 올린 요소 선택 : 선택자:hover{}

상태선택자

		입력 양식의 상태에 따라 선택되는 선택자
		1) 입력양식에 focus가 맞춰져 있는 경우  선택자:focus{}
		2) 입력양식이 체크 상태인 경우: 선택자:checked{}
		3) 입력양식이 사용 불가능한 경우: 선택자:disabled{}
		4) 입력양식이 사용 가능한 경우 : 선택자:enabled{}
