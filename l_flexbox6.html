2. item >>> flex-grow

		flex-grow: Container의 크기가 증가함에 따라 item의 크기가 증가할 때 증가하는 비율을 지정하는 속성
		* flex-grow:0 -> Container의 크기가 증가해도 item의 크기가 증가하지 않는다.
		* flex-grow:1이상 -> item의 크기와 상관없이 Container를 가득 채우도록 크기가 증가한다.

3. item >>> flex-shrink

		flex-shrink: Container의 크기가 감소함에 따라 item의 크기가 감소할 때 감소하는 비율을 지정하는 속성
		* flex-shrink:0 -> Container의 크기가 감소해도 item의 크기가 감소하지 않는다.
		* flex-shrink:1이상 -> Container의 크기가 item의 크기보다 작아질 때 item의 크기가 Container에 맞춰 함께 감소

4. item >>> flex-basis

		* flex-basis : item의 기본 크기를 결정하는 속성. default : auto
		* flex-basis를 사용해 item의 크기를 지정할 때는 모든 단위(px,%,em,vw,vh,vmin,vmax...)를 사용가능
		* flex-basis의 속성값을 0으로 지정하면 해당 item은 절대적 item(absolute flex item)이 되어
		 Container의 크기를 기준으로 item의 크기를 결정한다. -> 현재는 해당 요소의 최소너비를 지정
		* flex-basis를 auto로 지정하면 item은 상대적 item(relative flex item)이 되어 컨텐츠의 크기를 기준으로 크기가 결정
1
flex-basis:0
2
flex-basis:auto
3
flex-basis:400px
