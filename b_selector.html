CSS 선택자

		선택자란?
		
		특정 HTML Element를 선택하기 위해 사용하는 기능으로, Element를 선택해 원하는 스타일을 
		적용할 수 있게끔 해준다.
		
		선택자의 범위가 좁을 수록 우선순위는 올라간다.
		만약 선택자의 같다면 더 아래에 작성된 스타일이 적용된다.

전체 선택자

모든 태그를 선택할 때 사용하는 선택자
  • 테스트1
  • 테스트2
  • 테스트3

태그 선택자

		태그 이름으로 요소를 선택할 때 사용하는 선택자.
  • 태그선택자1
  • 태그선택자2
  • 태그선택자3

class 선택자

		class 속성의 속성값으로 원하는 element를 선택하는 선택자
  • 클래스선택자1
  • 클래스선택자2
  • 태그선택자3

id 선택자

		id 속성의 속성값으로 원하는 element를 선택하는 선택자	
		id 속성값은 해당 문서에 고유한 값이어야 한다.
  • 아이디선택자1
  • 클래스선택자2
  • 태그선택자3
