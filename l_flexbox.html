flex container

		Container는 정렬할 요소들을 감싸는 부모요소.
		display:flex/inline-flex  지정을 통해 해당 요소를 Container요소로 만들 수 있다.
		Container 요소 안의 요소들을 item이라 부른다.
				
		display:flex : block 요소의 특징을 가지는 Container
		display:inline-flex: inline 요소의 특징을 가지는 Container

1. display:flex

2. display:inline-flex

		Container 안의 item들을 정렬할 방향을 지정할 수 있다.
		Container 요소의 flex-direction 속성에 row/row-reverse/column/column-reverse 값을 지정
		이때 flex-direction 속성으로 지정된 정렬 방향을 주축 이라 한다.
		주축과 교차되는 방향의 축을 교차축이라 한다.
		row : 좌->우 *default
		row-reverse : 우 -> 좌
		column: 상->하
		column-reverse: 하->상
