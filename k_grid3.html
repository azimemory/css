position : Element가 배치되는 위치를 지정하는 속성

		static: 기본값, 브라우저가 html요소를 배치하는 위치에 요소가 배치된다.
				top,bottom,left,right,z-index 속성을 사용할 수 없다.
				
		relative: 브라우저가 html요소를 배치시키는 위치에 요소가 배치된다.
				top,bottom,left,right,z-index 속성을 사용할 수 있다.
				
		absolute: 부모요소의 좌측상단에 요소가 배치된다.
				top,bottom,left,right,z-index 속성을 사용할 수 있다.
				top,bottom,left,right 속성은 relative와 다른 기준으로 동작한다.
				position이 absolute인 요소는 normal flow에서 제외된다.
				
		fixed: 스크롤과 상관없이 화면 좌측상단을 기준으로 요소가 배치된다.
				top,bottom,left,right,z-index 속성을 사용할 수 있다.

top/bottom/left/right

		top: 속성값으로 지정한 크기만큼 요소를 위에서 아래로 이동
		bottom : 속성값으로 지정한 크기만큼 요소를 아래에서 위로 이동
		left: 속성값으로 지정한 크기만큼 요소를 왼쪽에서 오른쪽으로 이동
		right: 속성값으로 지정한 크기만큼 요소를 오른쪽에서 왼쪽으로 이동
	
		position:static  동작 x
		position:relative  요소의 현재 위치를 기준으로 이동
		position:absolute  부모요소의 top/bottom/left/right 를 기준으로 이동
