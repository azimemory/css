block-level / inline-level elements의 영역과 크기

1. block/inline element의 영역

block level element : block요소는 너비가 부모 elements의 너비와 동일하고
높이는 컨텐츠의 높이와 동일하다.
block level element : block요소는 너비가 부모 elements의 너비와 동일하고
높이는 컨텐츠의 높이와 동일하다.
inline level element : inline요소는 너비가 컨텐츠의 너비와 동일하고 높이도 컨텐츠의 높이와 동일하다. ***추가 elements***

2. block 요소와 inline 요소의 hight, width

block요소는 width속성과 height 속성을 사용할 수 있다.
width:30% / height:200px
inline요소는 width 속성과 height 속성을 사용할 수 없다. width:300000%/height:200000px

3. block요소와 inline요소의 margin, padding

	 	margin: 테두리(border)를 기준으로 다른 요소와의 거리
	 			
	 			**마진상쇄: 요소간 margin-top과 margin-bottom 영역이 겹칠 경우 두 요소 중 margin값이 큰 요소의 
	 			margin만 적용  
	 			
	 			
	 	padding: 테두리(border)를 기준으로 컨텐츠와의 거리
block요소의 margin은 상하좌우 모두 적용 된다.
margin-top:3%;
margin-bottom:3%;
margin-left:3%;
margin-right:3%;
block요소의 margin은 상하좌우 모두 적용 된다.
margin-top:3%;
margin-bottom:3%;
margin-left:3%;
margin-right:3%;
