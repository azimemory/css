Container >>> justify-content 속성

		justify-content : 주 축의 정렬 방식을 설정
		justify-content : flex-start >> items를 시작점부터 정렬 (default)
		justify-content : flex-end >> items를 끝점 부터 정렬
		justify-content : center >> items를 가운데 정렬
		justify-content : space-around >> items를 일정 간격으로 정렬
		justify-content : space-between >> items를 일정 간격으로 정렬, 시작/끝 요소를 시작점과 끝지점에 배치

1.justify-content : flex-start

2.justify-content : flex-end

3.justify-content : center

4.justify-content : space-between

5.justify-content : space-around
